Разработал веб-приложение на Python и Flask для генерации случайных фактов.
Реализовал backend-логику с HTTP-маршрутами и API-эндпоинтом, который возвращает данные в JSON.
Сделал frontend-шаблон с загрузкой новых данных без перезагрузки страницы через JavaScript fetch.
Добавил счётчик просмотров через Flask session и функцию копирования текста в буфер обмена.
Подготовил проект к запуску в контейнере с помощью Docker.
