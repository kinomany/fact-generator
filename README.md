Простое веб-приложение на Python (Flask), которое показывает случайные факты и позволяет получать новые без перезагрузки страницы.

Проект сделан как учебный, но реализует базовые вещи backend-разработки: работа с HTTP, API, шаблоны, взаимодействие frontend ↔ backend и контейнеризация через Docker.
